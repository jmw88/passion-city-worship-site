## Cancellations / Sick / Running Late

When you are scheduled, please know we are counting on you and asking that you make every effort to be on time. If you happen to ever run late, contact the point person for that environment (they will be the one to send you info closer to your rostered dates) + the worship leader or Music Director (MD) for the environment you're leading in for that day.

If you are feeling sick, contact your environment point person as soon as possible so we can find a replacement for that day.

You are placed in a very unique leading role, so **last minute cancellations are not acceptable** with the exception of emergencies, unforeseen events, and illnesses.

### TRL Live Walk-Thru

- **Check-in:** You will receive a temporary badge at security to get you through the doors for the day.
- **Check-out:** When leaving, return the badge at security each week (these belong to TRL LIVE). If security is not present, it is fine to just leave it there.
